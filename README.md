# n8n RealEstate Lead Workflow

## Description
This n8n workflow captures leads from a webhook or chatbot and processes them for storage or notifications.

## How to use
1. Download the JSON file
2. Open n8n
3. Click Import workflow
4. [Import into n8n](https://n8n.io/workflows/import)
5. Upload the JSON file
6. Configure credentials and webhook URL

## Requirements
- n8n v1.x or higher
